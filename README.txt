== Installation ==

1. Upload `wpcalc-ceiling` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the post, you will see the icon 'Wpc Ceiling' to select the calculator

The calculator is hosted by using the shortcode [wpcalc-ceiling]
